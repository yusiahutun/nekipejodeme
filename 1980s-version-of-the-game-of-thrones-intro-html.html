Someone Made an 80s Retro Version of the Game of Thrones Intro

Posted by Reinaldo Massengill on Sunday, July 21, 2024

YouTuber Mikolaj.Birek took a vintage version of the Game of Thrones intro by hunterlsanders and a retro-remixed version of the theme song by Steve Duzz and put them where they belonged: together. Now we have what might be the closest we’ll ever get to imagining what it might look like if Game of Thrones aired in the ‘80s. Sadly, there’s nothing we can do about Tyrion’s hairdo. Next time, someone needs to add some mullets in here.
